블루투스 이어폰 사용 후기

  • 휴대성이 좋아서 실외에서도 사용하기 용이함
  • 가격이 저렴하여 구매하기 쉬움
  • 디자인이 깔끔하고 세련되어 보기 좋음
  • 배터리가 내장되어 있지 않아 더 가벼워서 휴대하기 편함
  • 유튜브에서 가성비가 좋다는 평가를 받음

무선 이어폰 사용 후기

  • 노트 9와 아이폰10이 연결 안돼서 불편함을 호소함
  • USB-C 형 jack이 제품에 꽂힌 채 앞 부분이 빠지는 등의 제품 결함 발견
  • 제품을 두 개 샀지만 둘 다 무용지물처럼 사용하지 못하여 부담스러움을 나타냄
  • 영화도 못 보고 성탄절을 기분 나쁘게 보냄
  • 제품을 잘 쓰면 좋은 점이 있다고 하지만, 사용하지 못한다면 다음부터 이런 제품은 살지 말아야 겠다고 생각함

모바일 모니터 사용 후기

  • 가성비가 좋다
  • 휴대용이라 휴대성이 좋다
  • C to C 케이블 하나로 노트북, 휴대폰, 닌텐도와 연결 가능
  • 내장 스피커가 쓸만하다
  • 모니터 거치대가 있어 사용이 편리하다

닌텐도 스위치 덱스모드 사용 후기

  • 닌텐도 스위치 덱스모드 가성비가 좋음
  • 기본 케이블 어댑터만으로도 충분한 정상 작동
  • 내구성이 좀 떨어져서 고장 날 가능성 있음
  • 덱스모드 사용시 모니터 스피커 소리는 작게 들림
  • 스위치로 연결 후 모니터 스피커 소리는 잘 들림

모바일 모니터 설치 후기

  • 백라이트 100으로 조정 시 노트북 화면과 유사한 사용감 가능
  • 비활성화된 경우 Type-c PD 쪽 제일 하단쪽 Type-c에 꽂으면 해결 가능
  • 노트북 키보드 쪽에 올려놓고 사용 가능하여 인강, 코딩, 문서 작성 가능
